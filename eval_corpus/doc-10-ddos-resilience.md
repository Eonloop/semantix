# DDoS Resilience for Public Services

Distributed denial-of-service attacks target availability through traffic floods.

Resilience layers:
- CDN and edge filtering for volumetric attacks.
- Autoscaling and queue buffering for traffic spikes.
- Upstream coordination with ISP mitigation services.

Readiness practices:
- Define service degradation modes.
- Test playbooks with game days.
- Monitor latency and error budget burn in real time.

Availability defense should be measured as recovery time and user impact.
