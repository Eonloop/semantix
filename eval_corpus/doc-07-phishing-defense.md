# Enterprise Phishing Defense Program

Phishing remains a leading initial access method for attackers.

Program components:
- Employee awareness training with simulations.
- Email authentication using SPF, DKIM, and DMARC.
- Conditional access for suspicious sign-ins.

Detection and response:
- Fast user reporting channels.
- Automated mailbox triage.
- Credential reset and session revocation workflows.

Reducing click rate is helpful, but response speed is equally important.
